Content-Aware Image Resizer
===========================
An advanced content-aware, image resizing program utilizing seam carving algorithm in C++ that allows image to be scaled without losing/distorting meaningful image landmarks. It is an implementation of the seam carving algorithum described in [Avidan and Shamir (2007)](http://graphics.cs.cmu.edu/courses/15-463/2007_fall/hw/proj2/imret.pdf)
